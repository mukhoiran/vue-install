All about VueJS

Definitions

Javascript framework that is prioritized for building user interfaces in MVC-based applications

Benefits :

  • Approachable : Only need to understand HTML, CSS, and Javascript basic
  • Maintainable : Easy to maintain
  • Versatile : Completed library and fiture
  • Reactive : can adjust data changes
  • Performant : fast performance on small size
  • Testable : easy on testing

Requirement

  • Understand HTML
  • Understand CSS
  • Understand basic Javascript

Installation :

  • Only embed CDN on html header like Jquery
  • CDN : https://cdn.jsdelivr.net/npm/vue
